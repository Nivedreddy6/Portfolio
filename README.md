# 🚀 3D Portfolio

> A modern, interactive personal portfolio website built with React and Vite, featuring smooth scroll animations, an animated background, and an integrated contact form.

This repository contains the source code for a dynamic portfolio website designed to showcase projects, skills, and personal information with a highly polished and professional user experience. 

## ✨ Features

- **Modern Tech Stack**: Built with React 19 and Vite for extremely fast development and optimized production builds.
- **Engaging UI**: Implements smooth, native scroll-based reveal animations using the browser's `IntersectionObserver` API.
- **Dynamic Backgrounds**: Features a custom animated background with a subtle grid overlay for a "tech/3D" aesthetic feel.
- **Sectioned Layout**: Clean, single-page professional layout divided into Hero, About, Skills, Projects, and Contact sections.
- **Active Navigation**: Navigation links intuitively highlight as the user scrolls to their respective sections.
- **Responsive Design**: Designed to look and function perfectly across all device sizes (mobile, tablet, and desktop).

## 🛠️ Setup & Installation

To run this project locally:

1. **Clone the repository:**
   ```bash
   git clone <your-repository-url>
   cd 3d-portfolio
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. **Build for production:**
   ```bash
   npm run build
   ```

## 📂 Project Structure

- `src/components/` - Dedicated functional React components (`Hero`, `About`, `Skills`, `Projects`, `Contact`).
- `src/App.jsx` - Main application shell containing the scroll-observation logic and structural layout.
- `src/index.css` / `src/App.css` - Global styling, CSS variables, and layout systems.
- `old_vanilla/` - Previous vanilla HTML/CSS version of the portfolio.
